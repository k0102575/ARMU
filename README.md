## 서비스 소개
* 음악이 필요한 곳엔 음악을, 일이 필요한 뮤지션들에게 일거리를 연결해주는 Arbeit for Musicians (ARMU)입니다.<br><br>

## 서비스 목표
* 일반인과 뮤지션 간에 행사 직거래 서비스를 제공하는 것<br><br>
